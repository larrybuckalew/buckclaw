ARCHITECTURE HIGHLIGHTS
=======================

Flow: User Input → ChatLoop → CommandRegistry.dispatch()
      ↓ (if "/")
      Command.execute() → Response printed, continue loop
      ↓ (if not "/")
      AgentSession.chat() → LLM call → Response printed

Design Decisions:
- Commands parsed before LLM to exclude from context
- Registry injected into session for command access
- Skill loading deferred to SkillsCommand (lighter initialization)
- Session metadata optional (commands handle missing data)
- Type checking guards prevent circular imports
